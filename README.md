A simple Laravel package that provides a middleware which will require users to confirm routes utilizing their pin for authentication.

## REQUIREMENTS

- PHP 8.0+
- Laravel 10+

## STEPS TO INSTALL

``` shell
composer require ikechukwukalu/cloudspaceaml
```
